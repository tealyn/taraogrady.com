Girls Who Code Club Site

Overview

This is a current project that was started as a collaboration during a hack-a-thon. This is a work in progress, but will soon serve as a site for the Girls Who Code chapter in Oswego to post their projects!

I have been a facilitator for Girls Who Code for since 2017, and love seeing what these young women are capable of. Soon you will be able to see what they've accomplished here!
